# Better OmniRoute

A personal fork of OmniRoute. My gateway, my way.

## Changes so far

- **Strict-random strategy** — shuffle deck rotation for models and connections. No repeats until every option is used.
- **API key controls** — allowedConnections, autoResolve, is_active toggle, access schedules, key renaming.
- **Connection groups** — environment grouping in the Limits page with accordion UI.
- **Limits page UX** — persistent state, smart defaults, localStorage.
- **autoResolve** — bare shorthands (sonnet, haiku, opus) resolve to Claude automatically.
- **29 new tests** across all new features.
- **i18n** — 30 languages updated.
- ...and probably whatever I think of next.
